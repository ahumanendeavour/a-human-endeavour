This site is...

This site is a hodgepodge of ideas, inspirations, and chunks of code butchered from other projects, including some of my own. I've made every effort to catalogue what I've used here. No doubt this list will expand as the site does.


Technical Resources

The JS coding that enables the post lists, the picture galleries, and the header/footer injection, as well as the CSS code that makes them all look pretty, came from...

  • Zonelets
  • ZoneLots
  • The Bumblebee Framework

All of these projects have been massively helpful in implementing incredibly useful features, that would have been far beyond me at my current level of understanding.

  • Aseprite For the few bits of pixel-art.
  • Pictures taken by me unless credited otherwise.
  • Netlifyfor hosting.
  • VSCode Specifically the October 2024 build because I updated my software half way through making this and it decided my OS was too old and wouldn't let me use it at all anymore.
  • Koder for IPhone for whipping up entries in html on the go.
